# EmailDiet Professional Design System v1

## Vision
EmailDiet is a Mailbox Intelligence Platform focused on clarity, premium UX, trust, and actionable insights.

## Product Principles
- Insights before data
- Actions before reports
- Progressive disclosure
- Calm interface
- Accessibility first
- Motion with purpose

# Brand Personality

- Intelligent

- Human-centered

# Motion

Hover: 120–180ms

Layout: 200–250ms

Page: 300ms

Charts: 500ms

Counters: 600ms

Use subtle scale, elevation and fade transitions.
